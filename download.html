Give it a try ...

The Enna project has not yet been officially released (but that should come pretty soon). If you wanna give it a try, you probably need to consider yourself a developer and install the mandatory dependancies.

Download Sources

At GeeXboX, we use the Mercurial project as SCM for source control.

It's actualy pretty easy to retrieve current sources for Enna, through the following:

hg clone http://hg.geexbox.org/enna

If you already done so, updating the sources to latest development tree can be done quite easily through:

hg pull -u
